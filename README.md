### To get the project to work:
Install Parcel using Yarn or npm:
```
Yarn:
yarn global add parcel-bundler
```
 
```
npm:
npm install -g parcel-bundler
```

Run Parcel: 
```
parcel index.html
```
### Technologies used:
- JavaScript
- Three.js
- WebGL (shader)
- 3dsMax (for dino modelling)
- Babylon.js (for .glb export from 3dsMax)
